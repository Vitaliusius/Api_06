# Отправка комиксов в Telegramm-канал

Этот скрипт публикует один случайный комикс 'xkcd Рэндела Манров' в Telegramm.

## Запуск скрипта

Вам потребуется API token вашего бота телеграмм, а так же chat id вашего Telegram-канала. Так же нужно сделать вашего бота администраторм вашего канала. API token вашего бота присвоить переменной "TLGRM_BOT_API_KEY", chat id вашего канала присвоить переменной "TLGRM_CHAT_ID" в файле '.env' как в примере ниже:

```
TLGRM_BOT_API_KEY=sdf6sd7d7d7dfkservbnvi
TLGRM_CHAT_ID=@channel_id
```

Cкрипт запускается команой:

```
python main.py
```

## Окружение

Python3.14.2 уже должен быть установлен. Для изоляции проекта рекомендуется использовать virtualemv/venv. Для установки зависимостей:

```
pip install -r requirements.txt
```
